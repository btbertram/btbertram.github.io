Project Keystone

A small puzzle-action game made in Unity, where the player shifts rows and columns of different tiles to arrange them in specific patterns to clear them away. This project is made on my own time, for fun! The technology is simple, implementing a basic search system to find matching patterns in the play area. A short clip of the first playable build can be see here.

Yoink! Museum Heist

A small party-game made in Unreal Engine 4 as a university project. As part of team of nineteen people, we worked together to create this game in the timeframe of about four months. As one of the three engineers, I was tasked with creating backend game systems, helping manage source control, and sometimes debugging/optimizing blueprints created by the designers. I also occasionally provided design input which made it into the final game. A video showcase of the game can be seen here.

EggsCompany

A turn-based tactical game prototype heavily inspired by XCOM: Enemy Unknown, made in Unity, as a university project. As part of a team of three people, we were tasked with creating a game that showed off several AI programming techniques. My role was to implement pathfinding, and the game implements an A* pathfinding algorithm. Part of this also required implementation of a search system, which was also robust enough to be able to implement other searches, able to provide elements of environmental information to the player and AI necessary for decision making. A short clip of this system in action can be seen here.

Gatherin’ Game

A technical demo made in Unity with SQLite, which served as my final university project. This project was aimed at researching and implementing an external database as a backend system in a game. The game includes several gameplay features through this system, including world and player persistence, achievements, leaderboards, and additional features required to facilitate these, such as a user account system and concurrent programming techniques to avoid application locks while queries were run. The entire project can be seen on a github repository here.
